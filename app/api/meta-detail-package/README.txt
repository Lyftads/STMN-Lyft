Pacchetto pagina Meta dettagliata

Carica i file mantenendo questa struttura:

app/api/meta-detail/route.js
app/meta/page.js

Dopo il deploy, la nuova pagina sarà disponibile su:

/meta

Variabili ambiente richieste:
META_ACCESS_TOKEN
META_AD_ACCOUNT_ID

Nota:
META_AD_ACCOUNT_ID può contenere uno o più account separati da virgola.
